Utiliser git

Fichier créer pour tester les commandes git.

les commandes indispensables de git:

  1. git init : pour initialiser un noveau repository
  2. git log : pour s'indentifier auprés de Github
  3. git add . : afin que les modifications soient prisent en compte
  4. git commit -m "mentionner les modifications approtées": pour enregistrer en local votre trvail, il n'écrase pas les sauvegardes précédentes
  5. git push origin nomDelabrache : récupére les précédentes commit pour les mettre en ligne, ils seront alors accessibles depuis Github
  6. git remote add origin https://github.com/AHMADZAI-Niamatullah?tab=repositories : lorsque le fichier à déjà été crée en local et qu'on à créé un repo sur Github pour le stocker (ne rien mettre de dans avant de taper cette ligne )
